# Database Engineering Portfolio

This portfolio represents a complete body of work in database design, SQL development, and database administration using Microsoft SQL Server.

## Overview

This project consolidates multiple database assignments, design projects, and real-world scenarios into a single structured portfolio. It demonstrates the ability to design, implement, and manage relational databases from the ground up.

The work spans database architecture, SQL programming, security, performance optimization, and administrative operations.

## What This Portfolio Includes

### Database Design
- Fully normalized relational database models
- Entity relationship diagrams (ERDs)
- Data dictionary and naming standards
- Real-world database scenarios (restaurant, animal system)

### SQL Development
- Complex queries using joins, aggregations, and filtering
- Views and stored procedures with parameters
- Control-of-flow logic (IF, ELSE, CASE)
- Data manipulation and transformation

### Advanced SQL & System Logic
- Triggers for auditing and cascading deletes
- Error handling and logging systems
- Temporary tables and CTEs
- Subqueries and UNION operations

### Database Administration (DBA)
- Backup and recovery planning and execution
- Active session monitoring and termination
- User and role management
- Security policy design

### Performance & Optimization
- Index creation and performance comparison
- Query execution analysis
- Large dataset handling
